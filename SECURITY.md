# Security Policy

## Supported Versions

Only the latest main branch is currently supported for security updates.

| Version | Supported |
| ------- | --------- |
| 1.x     | ✅ Yes    |
| < 1.0   | ❌ No     |

## Reporting a Vulnerability

We take the security of FanOps Unified seriously. If you find any security vulnerability, please do NOT report it publicly via issue reports or pull requests. 

Instead, please send an email to security@fanops.dev.

We will acknowledge receipt of your report within 48 hours and work with you to analyze and resolve the issue. Once patched, we will publish a security advisory.
